Raspagem Web - Domínio de email

O software foi desenvolvido PuppeteerSharp, uma ferramenta de automação web via browser, que é uma adpatação do Node JS para o C# Usando o PuppeteerSharp conseguimos algumas vantagens sobre o Selenium, como velocidade de inicialização e navegação Sua principal finalidade é a raspagem de dados da web, nesse caso, informações de propriedades de domínios Nesse modelo, é feito a verificação de domínios ".br" e checamos todos dados vinculados ao mesmo Os resultados são todos escritos diretamente no banco de dados
